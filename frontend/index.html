// how it works

Send intent. Get results.

01
Send a natural language intent
No params, no schemas, no API keys. Just what you want done. Your agent sends a single GET request.
02
XoggAI searches and ranks endpoints
Vector search + LLM reranking across thousands of rated x402 APIs. Best endpoint wins — automatically.
03
402 payment negotiation
Server returns price and payment address. Nothing charged yet. Full transparency before commitment.
04
Agent pays and executes
USDC payment attached. XoggAI relays to upstream, returns data. Entire flow: autonomous, no human touch.
05
Result rated. Index improves.
AI scores 0–5. Bad endpoints sink, good ones rise. Every call makes the system smarter.
xoggai-agent.ts
// XoggAI — one endpoint, everything
import { XoggAI } from 'xoggai-sdk'

const agent = new XoggAI({
network: 'base-mainnet',
budget: 0.10 // USDC
})

// plain-English intent
const result = await agent.execute({
intent: "get ETH price, buy if <$3000",
autonomous: true
})

// routed → paid → executed → rated
console.log(result)
// { price: 2847.33, tx: "0xa3f...",
// rating: 5.0, latency: "11ms" }
